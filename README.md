# 炒股模拟器

一个本地运行的股票历史行情回放与交易模拟器。页面可以缓存行情数据，选择股票和历史时间点，只展示当时已经发生的 K 线数据，然后进行买入、卖出、不动和快进模拟。

## 功能

- 本地行情仓库：使用浏览器 IndexedDB 保存下载或导入的数据。
- K 线回放：支持日线、分钟线等 Yahoo Finance 支持的周期。
- 交易模拟：支持买入、卖出、不动、最大可买、按 K 线或时间快进。
- 账户计算：现金、持仓、市值、总资产、已实现盈亏、浮动盈亏。
- 费用设置：佣金、最低佣金、卖出印花税、滑点，可按 A 股/美股/港股模板切换。
- 公司行为：会根据数据源返回的分红和拆股事件调整账户。
- 图表：K 线、成交量、MA5/MA10/MA20/MA60、RSI、MACD、资产曲线。
- K 线交互：支持鼠标滚轮缩放、拖拽平移、30K/60K/120K/240K/全部等显示尺度。
- 辅助判断器：根据价格与 MA5、MA10、MA20、MA60 的关系，分类当前走势状态。
- 数据迁移：支持仓库 JSON 导入/导出，也支持 CSV 导入。

## 文件说明

- `炒股模拟器.html`：主程序，单文件网页应用。
- `stock_server.py`：本地行情下载助手，代理 Yahoo Finance 数据接口。
- `启动炒股模拟器.bat`：Windows 一键启动脚本。
- `data/datasets/`：通过启动脚本运行时，下载后的行情数据会保存在这里，每个数据集一个 JSON 文件。

## 使用方法

### 推荐方式：可在线下载行情

1. 在 Windows 中双击 `启动炒股模拟器.bat`。
2. 等待浏览器自动打开 `http://127.0.0.1:8765/`。
3. 点击“进入模拟”或“开始模拟”。
4. 输入股票代码，例如 `AAPL`、`MSFT`、`NVDA`、`600519.SS`、`000001.SZ`。
5. 选择 K 线周期和下载范围，点击“下载到仓库”。
6. 选择起始时间，点击“开始 / 重置”，开始买卖和快进。
7. 查看“辅助判断器”面板，参考均线排列、价格距离、趋势分类和缠绕状态。

运行时请保持启动脚本打开的命令行窗口不要关闭，否则网页无法继续在线下载行情。

通过这种方式下载的数据会保存到当前程序目录下的：

```text
data/datasets
```

如果程序放在 `Z:\Wang Junjie\自己学习\stock`，那么数据会保存在：

```text
Z:\Wang Junjie\自己学习\stock\data\datasets
```

## 辅助判断器说明

辅助判断器把均线系统当成“市场状态分类器”，不会给出确定预测。它会综合：

- 价格是否站上或跌破 MA5、MA10、MA20、MA60。
- MA5、MA10、MA20、MA60 是否形成多头排列或空头排列。
- 均线方向是向上、向下还是走平。
- 短期均线与中期均线是远离、靠近，还是反复交叉缠绕。

常见输出包括：

- 强势上涨：价格在主要均线上方，均线多头排列。
- 弱势下跌：价格在主要均线下方，均线空头排列。
- 震荡缠绕：MA5、MA10、MA20 距离很近或反复交叉。
- 上升趋势回踩：短期均线靠近 MA20，但中期趋势尚未破坏。
- 下降趋势反抽：短期均线靠近 MA20，但未能有效突破。
- 可能转强 / 可能转弱：长期趋势后出现缠绕并开始选择新方向。

这些结论只用于辅助复盘和分类，不构成买卖建议。

## K 线图操作

- 鼠标滚轮：缩放 K 线数量。
- 鼠标拖拽：左右平移历史窗口。
- `30K`、`60K`、`120K`、`240K`、`全部`：快速切换显示尺度。
- `最新`：回到当前模拟时间点。

### 直接打开 HTML

可以直接打开 `炒股模拟器.html` 使用演示数据、导入 CSV/JSON、进行模拟。

但由于浏览器对 `file://` 本地网页的联网限制，直接打开 HTML 时在线下载行情可能失败。此时数据会保存在浏览器自己的 IndexedDB 里，不是普通文件夹。需要下载行情并保存成可见文件时，请使用上面的启动脚本。

## 股票代码示例

- 美股：`AAPL`、`MSFT`、`NVDA`、`TSLA`、`AMZN`
- ETF：`SPY`、`QQQ`
- A 股：`600519.SS`、`000001.SZ`
- 港股：`0700.HK`

## CSV 导入格式

CSV 至少需要这些字段：

```csv
date,open,high,low,close,volume
2024-01-02,100,105,99,103,1234567
```

也支持部分中文字段名，例如 `日期`、`开盘`、`最高`、`最低`、`收盘`、`成交量`。

## 数据来源与限制

在线下载使用 Yahoo Finance Chart 接口。不同市场、周期和下载范围可能存在数据源限制，例如分钟级历史数据通常无法覆盖很久以前的时间。

本项目用于学习和模拟，不构成投资建议。实际交易规则、税费、分红到账、复权处理、交易时段和盘口成交可能更复杂，请以券商和交易所规则为准。

## 开发

这个项目不需要前端构建工具。修改 `炒股模拟器.html` 后刷新浏览器即可。

本地下载助手使用 Python 标准库实现，不需要安装第三方依赖。
